Scatterplot Matrix

If we want to show the relationships between n≥2 variables {xi}i=1n using Scatterplots, we need ∑i=1n∑j>in1=n(n−1)/2 scatterplots. If we plot xi∼xj and xj∼xi as two different plots, then we have n2−n plots. Then adding labels, we have a scatterplot matrix.

  • When passing multiple variables to the R built-in function plot, it will create a scatterplot matrix.
  • While this is quite useful for personal exploration of a dataset, it is not recommended for presentation purposes. Something called the Hermann grid illusion makes this plot very difficult to examine.

Other implementations

  • ggpairs() in GGally
  • splom() in lattice
